ubuntu 安装mysql #

  1. 更新软件包列表
shell
sudo apt update
1
  1. 安装 MySQL 服务器
shell
sudo apt install mysql-server
1

注意

安装过程中，系统可能会提示设置 MySQL 的 root 用户密码，请务必记住设置的密码。

  1. 启动 MySQL 服务
shell
sudo systemctl start mysql
1
  1. 检查 MySQL 服务状态
shell
sudo systemctl status mysql
1
  1. 进行安全设置
shell
sudo mysql_secure_installation
1

远程访问设置 #

  1. 登录 MySQL 服务器
shell
sudo mysql -u root -p
1

然后输入密码
2. 修改用户权限

shell
   GRANT ALL PRIVILEGES ON *.* TO 'your_username'@'%' IDENTIFIED BY 'your_password' WITH GRANT OPTION;
1

将 your_username 替换为您要用于远程访问的用户名，your_password 替换为对应的密码.
3. 刷新权限

shell
FLUSH PRIVILEGES;
1
  1. 修改 MySQL 配置文件
shell
sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf
1

打开文件后的编辑界面后操作步骤

  • 切换英文输入法，按下 I 键。
  • 找到 bind-address = 127.0.0.1 这一行，将其注释掉（在行首添加 # ）或者修改为您服务器的实际 IP 地址。
  • 按下Esc键。
  • 按下 shift + ; 分号组合键，然后输入wq，按下enter保存修改
  1. 重启 MySQL 服务
shell
sudo systemctl restart mysql
1

通过上面的步骤就可以在本地访问服务端的mysql服务了。
